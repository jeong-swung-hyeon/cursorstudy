# 한국 주식 가격 순위 대시보드

종가 기준 상위 50개 주식 정보를 보여주는 웹 대시보드입니다.

## 사용 방법

### 1. 서버 실행

Python 3가 설치되어 있어야 합니다.

```bash
# 서버 실행
python server.py
```

서버가 `http://localhost:8000`에서 실행됩니다.

### 2. 브라우저에서 열기

`index.html` 파일을 브라우저에서 열거나, 로컬 서버를 통해 접속하세요.

- 직접 열기: `index.html` 더블클릭
- 로컬 서버 사용 (권장): 
  ```bash
  # Python 3
  python -m http.server 8080
  # 그 다음 브라우저에서 http://localhost:8080/index.html 접속
  ```

### 3. 사용

1. 기준일자 선택 (기본값: 오늘)
2. "데이터 불러오기" 버튼 클릭
3. 상위 50개 주식 정보 확인

## 기능

- 종가 기준 상위 50개 주식 표시
- 시장구분 (KOSPI/KOSDAQ/KONEX) 표시
- 등락률 및 대비 정보 (색상 구분)
- 거래량 및 시가총액 정보
- 통계 정보 (전체 종목 수, 평균 종가 등)
- 반응형 디자인

## 파일 구조

- `index.html`: 메인 대시보드 페이지
- `server.py`: CORS 문제 해결을 위한 프록시 서버
- `README.md`: 이 파일

## 문제 해결

### CORS 오류가 발생하는 경우

`server.py`를 실행하여 프록시 서버를 사용하세요.

### API 인증 오류가 발생하는 경우

`index.html`과 `server.py`의 `API_KEY` 값을 확인하세요.
