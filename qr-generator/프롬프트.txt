1번
UI가 친절한 QR코드 제작 프로그램을 만들려고 할때 가장 추천하는 방법은 뭐야?

답변
가장 추천하는 방법: React + Vite + QR 라이브러리
추천 이유
빠른 개발: Vite로 빠른 개발 서버와 빌드
크로스 플랫폼: 웹 브라우저에서 바로 실행
배포 용이: Vercel, Netlify 등에 간단 배포
라이브러리 풍부: qrcode.react, qrcode.js 등
기술 스택
프레임워크: React + Vite (기존 프로젝트와 일관성)
QR 라이브러리: qrcode.react 또는 qrcode.js
UI 스타일링: Tailwind CSS 또는 CSS Modules
다운로드: html2canvas 또는 file-saver로 이미지 저장
주요 기능
텍스트/URL 입력
실시간 QR 코드 미리보기
크기 조절
색상 커스터마이징
다운로드 (PNG, SVG)
반응형 디자인

2번
최종으로 추천해준 기술스택으로 QR코드를 생성하는 프로그램을 만들어줘... 
- @현재폴더에 1.webp 이미지를 중앙에 넣고, 뒷 배경에 QR코드가 보이지 않도록 투병 배경 처리를 해줘.
- 링크는 복사하기 붙여넣기가 가능해야 하고, 처음에 1개만 입력하는 상태여야 해
- [+추가]버튼을 누르면 최대 10개까지 추가
- [생성]버튼을 누르면 한꺼번에 QR코드를 만들어서 @현재폴더에 저장
