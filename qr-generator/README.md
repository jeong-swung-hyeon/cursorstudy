# QR 코드 생성기

친절한 UI를 가진 QR 코드 생성 프로그램입니다.

## 기능

- 링크 입력 및 복사/붙여넣기 지원
- 최대 10개까지 링크 추가 가능
- 중앙에 이미지(1.webp)가 배치된 QR 코드 생성
- 투명 배경 처리로 QR 코드가 이미지 뒤에 보이지 않음
- 일괄 QR 코드 생성 및 다운로드

## 설치

```bash
npm install
```

## 실행

```bash
npm run dev
```

## 빌드

```bash
npm run build
```

## 사용 방법

1. 링크를 입력합니다
2. 필요시 "+ 추가" 버튼으로 최대 10개까지 링크를 추가할 수 있습니다
3. "생성" 버튼을 클릭하면 모든 QR 코드가 PNG 파일로 다운로드됩니다
